# Template Typescript + Prisma + Express

## Use Template
```
git clone https://github.com/Pedromdsn/Express-Prisma-Template
```

## Install yarn

```
npm i -g yarn
```

## Install Dependencies

```
yarn
```

## Execute Production

```
yarn start
```

## Execute Development

```
yarn dev
```

## To See tables

```
npx prisma studio
```

## To Update tables

```
npx prisma migrate dev --name dev
```
